# Smart Industrial Monitoring System

## Functional Requirements

FR-1
The system shall monitor environmental data.

FR-2
Each node shall publish telemetry through MQTT.

FR-3
The gateway shall store measurements.

FR-4
The dashboard shall display live node status.

FR-5
The system shall support OTA firmware updates.

FR-6
The system shall detect offline devices.

FR-7
The system shall support multiple sensor nodes.

---

## Non-functional Requirements


NFR-1
System shall recover after power loss.

NFR-2
Firmware shall run continuously for a week.

NFR-3
Gateway shall automatically restart after reboot.
